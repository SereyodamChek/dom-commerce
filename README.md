# 🛒 Dom Commerce – Next.js E-Commerce Platform

### 👨‍💻 Built by Sereyodam

---

## 📌 About This Project

**Dom Commerce** is a high-performance **e-commerce web application** built with **Next.js App Router**.

This project demonstrates modern full-stack commerce architecture using server-side rendering, optimized performance, and scalable design patterns.

---

## 🚀 Key Features

* 🛍️ Full e-commerce storefront UI
* ⚡ High performance with React Server Components
* 🔄 Server Actions for data handling
* 🎯 Optimistic UI updates
* 🔍 Advanced search capabilities
* 📱 Responsive modern design

---

## 🛠️ Tech Stack

* ⚛️ **Next.js (App Router)**
* 💻 **TypeScript**
* 🎨 **Tailwind CSS**
* 🧠 **React Server Components**

---

## 📂 Project Structure

```
/app         → Application routes
/components  → Reusable UI components
/lib         → Core logic & integrations
/fonts       → Custom fonts
```

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/dom-commerce.git
cd dom-commerce
```

### 2. Install dependencies

```bash
pnpm install
# or
npm install
```

### 3. Configure environment variables

Create `.env.local` from `.env.example` and configure required values.

---

## ▶️ Run the Project

```bash
pnpm dev
```

Open:
👉 [http://localhost:3000](http://localhost:3000)
